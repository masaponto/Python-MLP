# Python-MLP
Multi Layer Perceptron written in Python3.4.3  
batch learning is avilable  

## How to install 
I recommend you to install this to anaconda3-4.0.0  

```
pip install git+https://github.com/masaponto/python-mlp
```

## Require
- scikit-learn: 0.17.1
- numpy: 1.10.4
